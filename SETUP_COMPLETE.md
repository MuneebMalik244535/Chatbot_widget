## 🆘 Troubleshooting

### Issue: Port 3000 in use
```bash
npm run dev -- -p 3001
```

### Issue: Modules not found
```bash
npm install
```

### Issue: Styles not loading
```bash
# Delete cache
rm -rf .next

# Rebuild
npm run dev
```

### Issue: Changes not showing
- Hard refresh: `Ctrl+F5` (Windows) or `Cmd+Shift+R` (Mac)
- Clear browser cache
- Restart dev server
