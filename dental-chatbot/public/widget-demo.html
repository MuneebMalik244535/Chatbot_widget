🦷 Dental Clinic Widget System

Production-ready, embeddable chatbot widget for any website

✨ Key Features

🎯 Easy Integration

Add one script tag to your website. No setup required.

🚀 Multi-Tenant

Support multiple clinics with different data using clientId.

🎨 Customizable

Change colors, services, and doctor info easily.

📱 Responsive

Works perfectly on mobile, tablet, and desktop.

🤖 AI-Powered

Rule-based responses, fully frontend-based logic.

🔒 Secure

No backend server, no external API calls, fully isolated.

🚀 Quick Start

Add this one line to your website's HTML:

<script src="https://your-domain.com/widget.js" data-client="dental-clinic-1"></script>

That's it! Chat button appears instantly.

💻 Integration Examples

WordPress

Add to your theme's footer.php:

<!-- Dental Clinic Widget --> <script src="https://your-domain.com/widget.js" data-client="dental-clinic-1"></script>

Plain HTML Website

<!DOCTYPE html> <html> <body> <h1>Welcome to My Clinic</h1> <!-- Widget here --> <script src="https://your-domain.com/widget.js" data-client="dental-clinic-1"></script> </body> </html>

React/Vue/Angular

useEffect(() => { const script = document.createElement('script'); script.src = 'https://your-domain.com/widget.js'; script.setAttribute('data-client', 'dental-clinic-1'); document.body.appendChild(script); }, []);

🔗 API Reference

POST /api/mock-chat

Send a message and get a response:

Request:

{ "message": "What services do you offer?", "clientId": "dental-clinic-1" }

Response:

{ "success": true, "message": "We offer the following services: ...", "clientId": "dental-clinic-1", "clinicName": "Bright Smile Dental Clinic" }

💬 Chatbot Response Examples

  • User: "What services do you offer?"
    Bot: Lists all services with descriptions
  • User: "How much does cleaning cost?"
    Bot: Shows pricing for teeth cleaning
  • User: "When are you open?"
    Bot: Displays clinic hours
  • User: "How do I book an appointment?"

  • User: "Tell me about your doctors"
    Bot: Lists all doctors with experience

🎨 Customization

  • ⏰ Operating hours
  • 📞 Emergency contact

🏗️ Architecture Overview

External Website ↓ <script src="widget.js"> ↓ Creates iframe → /widget?clientId=clinic-id ↓ Loads WidgetChatbot component ↓ Chat → /api/mock-chat ↓ getClinicData(clientId) ↓ Returns clinic-specific response

❓ FAQ

Can I customize the widget colors?

Yes! Edit the theme property in clinicData.ts.

Can multiple clinics use the same widget?

Yes! Use different data-client values.

Do I need a backend?

No! Everything runs in the frontend.

Is it secure?

Yes! No external API calls, data stays in your domain.

Will it work on old browsers?

Works on all modern browsers (2020+).

How do I track analytics?

You can add Google Analytics easily by updating the code.
